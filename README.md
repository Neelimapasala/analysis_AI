> **Transforming Raw Data into Actionable Intelligence in Minutes**

Nebula AI is an end-to-end automated data analysis system that eliminates the need for manual data science expertise. Upload any CSV dataset and get instant insights, visualizations, ML predictions, and NLP analysis - all in one beautiful interface.

## ✨ Key Features

### 📈 Automated Data Processing
- **Smart Data Cleaning** - Auto-detect and handle missing values, duplicates, and outliers
- **Exploratory Data Analysis (EDA)** - Statistical summaries, correlation matrices, and distribution analysis
- **Pattern Recognition** - Automatically identify hidden patterns and trends

### 🤖 Machine Learning & Deep Learning
- **8+ Pre-built Models**:
  - Linear Regression
  - Ridge & Lasso Regression
  - Random Forest
  - SVR (Support Vector Regression)
  - KNN Regressor
  - Neural Network (Deep Learning)
  - XGBoost
- **Auto Model Training** - One-click model training with R² scoring
- **Feature Importance Analysis** - Understand what drives predictions

### 💬 Natural Language Processing (NLP)
- **Sentiment Analysis** - Understand text sentiment (Positive/Negative/Neutral)
- **Topic Modeling** - Discover themes in text data
- **Named Entity Recognition (NER)** - Extract people, organizations, dates
- **Text Summarization** - Auto-generate concise summaries

### 📊 Interactive Visualizations
- **Correlation Heatmaps** - Visual relationships between variables
- **Distribution Plots** - Understand data spread
- **Trend Analysis** - Spot patterns over time
- **Feature Importance Charts** - See what matters most

### 🤖 Nova AI Chatbot
- **Groq LLM Integration** - Ask questions in natural language
- **Context-Aware Responses** - Understands your specific dataset
- **Real-time Insights** - Get instant answers about your data
- **Smart Suggestions** - Ask about correlations, predictions, or cleaning

### 📄 Automated Reporting
- **Comprehensive Reports** - Export all analysis results
- **Model Performance Metrics** - R² scores and accuracy
- **Data Quality Summary** - Missing values, duplicates, data types
